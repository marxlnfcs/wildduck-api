    List registered DKIM keys

    see

    https://docs.wildduck.email/api/#operation/getDkimKey

    Parameters

    • dto: { description?: string; domain: string; privateKey?: string; selector: string }
      • Optional description?: string
        description

        Key description

      • domain: string
        description

        Domain name this DKIM key applies to. Use "*" as a special value that will be used for domains that do not have their own DKIM key set

      • Optional privateKey?: string
        description

        Pem formatted DKIM private key. If not set then a new 2048 bit RSA key is generated, beware though that it can take several seconds to complete.

      • selector: string
        description

        Selector for the key

    Returns Promise<{ description: string; dnsTxt: { name: string; value: string }; domain: string; fingerprint: string; id: string; publicKey: string; selector: string; success: boolean }>
